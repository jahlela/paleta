About Paleta

Hey, I'm Jahlela, the creator of Paleta. Here's a little FAQ about the project.

How do I make a palette?

Right click any image on the internet and click 'copy image address'. Paste this into the Paleta homepage and activate wizardry!

How does the palette generation work?

The app takes an image, breaks it down pixel-by-pixel, and uses a machine learning algorithm called K-Means Clustering to pick out the dominant colors in the image. You can see the code for that algorithm here.

Where did Paleta come from, anyway?

Paleta started as a final project while I was attending Hackbright Academy in SF. I was inspired to build it because I have often created color palettes for projects by hand (or used a site like Design Seeds). However, I wanted to build an app that would generate palettes automatically.

Can I fork it? Yes, you can!

Fork it, break it, twist it, turn it, just be sure to link to the original code if you do use it. :) Also, you may want to start with a fresh database and image library.

You can find the source code at https://github.com/jahlela/paleta
